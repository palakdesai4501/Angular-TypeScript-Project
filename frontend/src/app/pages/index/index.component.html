Objective:

Build an application to handle CRUD of JSON data

Requirement:

Front End:

Create at least 3 views

  • Data Table View:

    Show all the data in the 
    backend/data/funds_data.json
    in a table

  • User Facing View:

    Show the details of the data in a neat and orderly fashion

  • Admin Edit View :

    Interface where the user can edit these values and save back into the JSON file

    Note: do auto save whenever the user makes changes

Back End:

Create at least 3 APIs

  • Get all items (For Data Table View)

    Retrieve all data in the 
    backend/data/funds_data.json
    file.

  • Get single item (For User Facing View)

    Retrieve single item in the 
    backend/data/funds_data.json
    file.

  • Update item (For Admin Edit View)

    Update data in the 
    backend/data/funds_data.json
    file.
